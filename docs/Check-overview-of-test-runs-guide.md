## Overview

This guide provides step-by-step instructions on how to check the overview of test runs in Carrier. Carrier allows you to monitor and analyze the performance of your tests effectively. Follow these steps to access and review the test runs overview.

### Prerequisites

Before you begin checking the overview of test runs in Carrier, ensure that you have completed the following:

- Installed and set up Carrier successfully.
- Created a project in Carrier.
- Executed tests within the project.

### Steps

Follow the steps below to check the overview of test runs in Carrier:

### Step 1: Access the Carrier Web Interface

1. Open a web browser and enter the URL of your Carrier installation.
2. Log in to the Carrier web interface using your credentials.

> Note: Make sure you have logged in using the appropriate user account that has all required accesses.
{: .prompt-info }

### Step 2: Select the Project

1. From the project dropdown menu, select the desired project where you want to check the test runs overview.
![Select Project](/assets/posts_img/select_prj.png)
2. Click "Go!" to navigate to the project's configuration page.

### Step 3: Navigate to the Performance Tab

1. In the project configuration page, click on the "Performance" tab located in the left menu.
![Performance Dropdown](/assets/posts_img/performance_dropdown.png)

### Step 4: Check Overview of Test Runs

1. In the Performance section, you will see a list of test runs that have been executed within the selected project.
![Test Runs Overview](/assets/posts_img/test_runs_overview.png)
2. Each test run will display key information such as test name, execution status, start time, end time, and duration.
3. Click on a specific test run to view detailed results and metrics.
![Test Run Details](/assets/posts_img/test_run_details.png)

### Step 5: Analyze Test Run Results

1. In the detailed view, you can analyze various metrics such as average throughput, error rate, response time, and page speed.
![Test Run Metrics](/assets/posts_img/test_run_metrics.png)
2. Use the dropdown menus to filter and customize the displayed data.
3. Zoom in on graphs and charts for a more detailed view using the magnifying glass button.
![Zoomed Graph](/assets/posts_img/zoomed_graph.png)

### Step 6: Review Summary Tables

1. Scroll down to the bottom of the test run details page to review summary tables for backend and UI test results.
![Summary Tables](/assets/posts_img/summary_tables.png)

### Step 7: Export Test Run Results

1. If needed, export the test run results to Excel or PDF formats for further analysis or reporting.
![Export Results](/assets/posts_img/export_results.png)

By following these steps, you can effectively check and analyze the overview of test runs in Carrier, allowing you to monitor and improve the performance of your tests.
